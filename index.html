Machine Learning en el Dataset Titanic

Comparación de Modelos: Árbol de Decisión vs Random Forest

Este proyecto fue desarrollado por el grupo Fuerza Popular como parte del análisis de modelos de clasificación utilizando Machine Learning.

Problema Analizado

El dataset Titanic contiene información sobre los pasajeros que viajaban en el barco Titanic. El objetivo del proyecto es predecir si un pasajero sobrevivió o no al accidente utilizando técnicas de Machine Learning.

Este problema corresponde a un modelo de clasificación binaria.

Variable objetivo

Survived
1 = Sobrevivió
0 = No sobrevivió

Variables importantes

Sexo, Edad, Clase del pasajero, Precio del ticket, Puerto de embarque.

Preprocesamiento de Datos

Manejo de valores faltantes

Se reemplazaron valores faltantes en la variable Age utilizando la media y en Embarked utilizando el valor más frecuente.

Eliminación de columnas

Se eliminaron columnas irrelevantes como PassengerId, Name, Ticket y Cabin.

Codificación de datos

Las variables categóricas como Sex y Embarked se transformaron utilizando One-Hot Encoding.

División del dataset

Los datos fueron divididos en 80% entrenamiento y 20% prueba.

Modelo 1: Árbol de Decisión

El Árbol de Decisión clasifica los datos mediante una estructura jerárquica de reglas. Cada nodo representa una decisión basada en una característica del dataset.

Ventajas

  • Fácil de interpretar
  • Visualización clara de decisiones
  • Rápido de entrenar

Limitaciones

  • Puede sufrir overfitting
  • Sensible a cambios en los datos

Modelo 2: Random Forest

Random Forest es un modelo de ensamble que combina múltiples árboles de decisión. Cada árbol realiza una predicción y el resultado final se obtiene mediante votación.

Ventajas

  • Mayor precisión
  • Reduce el overfitting
  • Mejor generalización

Configuración

  • 100 árboles
  • Profundidad máxima controlada
  • Random State = 42

Métricas de Evaluación

Accuracy

Mide el porcentaje total de predicciones correctas.

Precision

Evalúa la exactitud de las predicciones positivas.

Recall

Mide cuántos sobrevivientes reales fueron detectados correctamente.

F1 Score

Balance entre Precision y Recall.

Curva ROC

Compara la tasa de verdaderos positivos y falsos positivos.

Conclusiones

Ambos modelos lograron predecir la supervivencia de los pasajeros del Titanic con buen desempeño. Sin embargo, el modelo Random Forest presentó mejores resultados debido a su capacidad de combinar múltiples árboles y reducir errores individuales.

El Árbol de Decisión permite entender fácilmente las reglas de decisión del modelo, mientras que Random Forest proporciona una mayor precisión y estabilidad en las predicciones.
