🎯 Problema Analizado

El dataset Titanic contiene información sobre los pasajeros que viajaban en el barco. El objetivo es predecir si un pasajero sobrevivió o no utilizando técnicas de Machine Learning. Corresponde a un modelo de clasificación binaria.

Variable objetivo

Survived
🟢 1 = Sobrevivió
🔴 0 = No sobrevivió

Variables importantes

Sexo, Edad, Clase del pasajero, Precio del ticket, Puerto de embarque.

⚙️ Preprocesamiento de Datos

Manejo de valores faltantes

Se reemplazaron valores en Age usando la media y en Embarked usando el valor más frecuente.

Eliminación de columnas

Se eliminaron columnas irrelevantes como PassengerId, Name, Ticket y Cabin.

Codificación de datos

Las variables categóricas (Sex, Embarked) se transformaron utilizando One-Hot Encoding.

División del dataset

Los datos fueron divididos en 80% entrenamiento y 20% prueba.

🌳 Modelo 1: Árbol de Decisión

Clasifica los datos mediante una estructura jerárquica de reglas. Cada nodo representa una decisión basada en una característica.

✅ Ventajas

  • Fácil de interpretar
  • Visualización clara de decisiones
  • Rápido de entrenar

⚠️ Limitaciones

  • Puede sufrir overfitting (sobreajuste)
  • Sensible a cambios en los datos

🌲🌲 Modelo 2: Random Forest

Es un modelo de ensamble que combina múltiples árboles de decisión. Cada árbol predice y el resultado final es por votación.

✅ Ventajas

  • Mayor precisión
  • Reduce el overfitting
  • Mejor generalización

🛠️ Configuración

  • 100 árboles
  • Profundidad máxima controlada
  • Random State = 42

📊 Métricas de Evaluación

Accuracy

Mide el porcentaje total de predicciones correctas del modelo.

Precision

Evalúa la exactitud de las predicciones positivas.

Recall

Mide cuántos sobrevivientes reales fueron detectados.

F1 Score

Balance armónico entre Precision y Recall.

Curva ROC

Compara la tasa de verdaderos positivos vs falsos positivos.

💡 Conclusiones

Ambos modelos lograron predecir la supervivencia con buen desempeño. Sin embargo, el modelo Random Forest presentó mejores resultados debido a su capacidad de combinar múltiples árboles y reducir errores individuales.

El Árbol de Decisión permite entender fácilmente las reglas, mientras que Random Forest proporciona una mayor precisión y estabilidad en las predicciones.
